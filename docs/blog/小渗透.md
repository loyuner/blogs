### 信息收集

学长是给我已经确定范围，替我省去了*查真实ip*这个步骤，我只用扫C段旁站子域名什么的就行

#### process1-C段

我喜欢使用nmap进行，接下来我列出基础使用方法供兄弟们

1. 扫描C段主机存活

```
Nmap -sn -PE -n 192.168.1.1/24  -oX out.xml

-sn  不扫描端口
-PE  ICMP 扫描
-n  不进行dns解析
```

2. 全端口扫描

```
Nmap -sS -Pn -p 1-65535 -n  ip

-sS  半开放扫描
-Pn  不进行主机存活探测
```
